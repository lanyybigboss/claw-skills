## Search Commands to Try

### GitHub API Searches
```bash
# Search for FenNAS
curl -s "https://api.github.com/search/repositories?q=fennas"

# Search for Iron Blade Technology
curl -s "https://api.github.com/search/repositories?q=iron+blade"

# Search for NAS projects
curl -s "https://api.github.com/search/repositories?q=nas+language:chinese"

# Search by topic
curl -s "https://api.github.com/search/repositories?q=topic:nas+topic:storage"
```
